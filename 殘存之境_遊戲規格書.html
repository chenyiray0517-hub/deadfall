《殘存之境 Deadfall》— 3D 喪屍生存遊戲 完整規格書

1. 遊戲概述

1.1 一句話簡介

病毒爆發後的世界,玩家扮演倖存者,在荒野、鄉村與城市之間搜刮資源、建造據點、對抗感染者(人類與動物),並在飢餓、口渴、體力與傷病的夾縫中活下去。

1.2 世界觀背景

  • 「NV-7 諾瓦病毒」於三年前爆發,透過體液與咬傷傳染,感染人類與哺乳類動物。
  • 感染者不是傳統的「死者復活」,而是失去理智、痛覺遲鈍、攻擊性極強的活體宿主 —— 因此他們會奔跑、會挨餓、會被聲音與氣味吸引。
  • 政府崩潰,城市成為感染重災區(物資最豐富、風險最高);鄉村半荒廢(中等資源、中等風險);荒野相對安全但物資稀缺、有感染野獸。
  • 玩家的長期目標可以是:找到傳聞中的「淨區」、研發抗病毒血清、或單純建立一個能自給自足的堡壘。

1.3 核心遊戲循環(Core Loop)

探索搜刮 → 帶回物資 → 製作/建造/烹飪 → 強化角色與據點 → 挑戰更危險的區域 → 回到探索
                ↑ 全程受 飢餓/口渴/體力/生命 四大數值壓力驅動 ↓

3. 核心生存數值系統(四大數值)

3.1 生命值 HP(Health)

屬性 設定
範圍 0 – 100
歸零 死亡(依模式決定是否永久死亡)
減少來源 感染者攻擊、墜落、飢餓/口渴歸零後每秒扣血、流血、疾病、中毒、極端溫度
恢復方式 繃帶止血後自然緩慢回復(需飽食度>60%)、醫療包、藥品、睡眠
特殊機制 HP < 30% 時畫面泛紅、移動速度 -15%、瞄準晃動加劇

3.2 飽食度(Hunger)

屬性 設定
範圍 0 – 100
自然消耗 每遊戲小時 -1.5(靜止)/-3(奔跑、戰鬥、砍伐等勞動)
歸零懲罰 每 5 秒 -1 HP,體力上限減半
低於 25% 體力恢復速度 -50%,無法自然回血
高於 80% 體力恢復 +25%,自然回血啟動
恢復方式 進食(生食有食物中毒風險,熟食安全且回復量更高)

3.3 口渴度(Thirst)

屬性 設定
範圍 0 – 100
自然消耗 比飽食度快:每遊戲小時 -2.5(炎熱天氣或奔跑時 ×2)
歸零懲罰 每 3 秒 -1 HP(比飢餓更致命)
低於 25% 視野邊緣模糊、體力消耗 +30%
恢復方式 喝水。髒水(河水、雨水桶)直接喝有 40% 機率染上痢疾;需煮沸或用淨水錠

3.4 體力值(Stamina)

屬性 設定
範圍 0 – 100,即時消耗即時恢復
消耗行為 奔跑(-5/秒)、跳躍(-10)、近戰揮擊(-8~15)、攀爬(-8/秒)、格擋(-12)、拉弓(-3/秒)
歸零 強制停止奔跑、無法揮武器、喘氣聲會吸引感染者
恢復速度 站立 +8/秒、行走 +4/秒;受飽食/口渴/負重/疾病影響
上限機制 長期營養不良會暫時降低上限;技能可永久提升上限

3.5 進階狀態(建議 M2 之後加入)

  • 體溫:淋雨、涉水、夜晚荒野會失溫;失溫時體力恢復停止、持續扣血。靠營火、穿衣保暖。
  • 感染值(Infection):被感染者咬傷/抓傷有機率累積感染值,0→100 後轉化死亡。用抗生素延緩、血清治癒(稀有)。
  • 疲勞/睡眠:超過 20 小時未睡眠出現視覺搖晃、體力上限 -30%。睡覺可快轉時間但據點不安全時可能被襲擊。
  • 負重:背包超重時移動變慢、體力消耗增加。
  • 傷病狀態列表:流血、骨折(需夾板,移速大減)、食物中毒、痢疾(口渴消耗×3)、感冒、中毒。

4. 世界與地區設計

4.1 世界結構

  • 地圖尺寸建議 4km × 4km(程序化生成 + 手工地標混合)。
  • 三大生態區以「風險/報酬」梯度排列:荒野(低風險低報酬)→ 鄉村(中)→ 城市(高風險高報酬)。
  • 動態天氣:晴、陰、雨(淋濕失溫)、雷暴(雷聲掩護玩家腳步,也刺激感染者躁動)、濃霧(視野縮短,潛行加成)。
  • 日夜循環:一天 = 現實 40 分鐘。夜晚感染者移動速度 +30%、偵測範圍 +50%、刷新量 ×2,鼓勵玩家夜晚據守、白天外出。

4.2 荒野區(Wilderness)

項目 內容
地形 森林、山丘、河流、湖泊、洞穴、沼澤
資源 木材、石頭、野果、草藥、獵物(鹿/兔/野豬)、淡水、蘑菇(部分有毒)
威脅 感染狼群、感染熊、毒蛇、失溫、迷路(無路標)
地標 護林員瞭望塔(望遠鏡+地圖碎片)、獵人小屋、洞穴系統(礦石+蝙蝠群)、墜毀直升機(軍用物資)
玩法定位 前期起步區、建立隱蔽據點的首選、食物與建材主要來源

4.3 鄉村區(Rural)

項目 內容
地形 農田、農舍、穀倉、小鎮街道、加油站、教堂、鄉間公路
資源 種子與農具、罐頭、燃料、工具、獵槍與彈藥、拖拉機/皮卡零件、井水
威脅 散布的感染農民、感染家畜(牛/豬/狗)、穀倉內的感染者巢穴
地標 大型農場(可整修為據點)、水塔(登高觀察)、鄉鎮警局(武器櫃,需撬鎖)、診所(醫療品)
玩法定位 中期核心:農耕起步、載具修復、據點升級材料

4.4 城市區(Urban)

項目 內容
地形 高樓、地鐵站、醫院、超市、購物中心、警察總局、軍方隔離區、下水道
資源 高級武器、稀有藥品(抗生素/血清材料)、電子零件、防彈裝備、大量罐頭
威脅 高密度感染者群(屍潮)、特殊變異體、建築倒塌區、敵對掠奪者幫派
地標 醫院(血清研究線索)、軍方隔離區(最高級戰利品+最強變異體)、地鐵網(快速移動但黑暗)、屋頂花園(安全休息點)
玩法定位 後期挑戰區、劇情推進、頂級裝備來源

4.5 特殊區域

  • 軍事檢查哨:地圖邊緣,有軍用卡車、槍械、防毒面具。
  • 研究設施(終局地城):病毒起源地,劇情終點,可製作血清。
  • 感染巢穴:各區隨機生成,清除後該區域感染者刷新率大降,並掉落稀有物資。

5.3 感染者 AI 規格

  • 感知系統:視覺(錐形視野,夜晚縮短/夜行種加長)+ 聽覺(槍聲>奔跑>行走>蹲行)+ 嗅覺(玩家流血時被追蹤)。
  • 狀態機:閒晃 → 起疑(朝聲源調查) → 追擊 → 攻擊 → 失去目標(搜索 30 秒) → 返回閒晃。
  • 群體行為:一隻進入追擊,半徑 20m 內同類跟隨;屍潮事件時全圖朝噪音點聚集。
  • 人類敵人(掠奪者):會用槍、找掩體、包抄,也會被感染者攻擊 —— 可誘導「三方混戰」。

  • 鐵管炸彈、汽油彈、煙霧彈(掩護撤離)。
  • 噪音誘餌:鬧鐘/收音機,引開感染者的核心潛行道具。
  • 捕獸夾:定身敵人並造成流血;也能抓兔子。
  • 尖刺陷阱、絆線警報(空罐+繩)、電擊柵欄(需電力)。

6.7 工具類

7. 遊戲系統與多種玩法

7.1 製作系統(Crafting)

  • 分級工作站:徒手 → 工作台 → 熔爐 → 縫紉機 → 化學台 → 軍械台,越高級能做的東西越多。
  • 配方透過「撿到的說明書/雜誌」解鎖(例:《野外求生月刊》解鎖燻肉架),鼓勵探索。
  • 範例配方:
  • 布條繃帶 = 布料×2
  • 自製弓 = 木棍×3 + 繩索×1(需工作台)
  • 燃燒瓶 = 玻璃瓶 + 酒 + 布料
  • 滅音器 = 鐵管 + 廢金屬×3 + 膠帶(需軍械台)
  • 血清 = 變異體樣本 + 抗生素×3 + 化學套件(需研究所化學台)

7.2 建造與據點系統(Base Building)

  • 兩種路線:① 空地自建(地基→牆→屋頂,木製/金屬/混凝土三級材質)② 佔領現有建築整修(封窗、加固門、清空感染者)。
  • 防禦設施:木刺牆、鐵絲網、瞭望塔、絆線警報、電擊柵欄、加固門(可上鎖)。
  • 機能設施:營火/火爐、雨水收集桶、菜園、雞舍(撿蛋)、燻肉架、儲物箱、床(存檔+快轉時間)、發電機(耗油)→ 電燈/冰箱/電動工具。
  • 據點襲擊事件:每 3~7 天隨機一波屍潮夜襲(規模隨遊戲天數成長),噪音大的據點更常被襲擊 —— 讓建造防禦有意義。

7.3 農耕與畜牧

  • 作物:馬鈴薯(耐旱)、胡蘿蔔、番茄、玉米、藥草;需澆水、施肥(腐爛食物堆肥),有生長階段模型。
  • 動物:捕捉未感染的雞/兔飼養;感染烏鴉會偷襲雞舍。
  • 季節(進階):冬季作物停產,逼迫玩家秋季囤糧。

7.4 狩獵、釣魚與採集

  • 狩獵:追蹤腳印/血跡,弓箭無聲狩獵不驚動感染者;剝皮取得肉+皮革+骨頭。
  • 釣魚:湖河海不同魚種,小遊戲式收竿判定。
  • 採集:草藥辨識(技能影響)、蜂巢採蜜、鳥巢取蛋。

7.5 載具系統

載具 特性
腳踏車 無聲、不耗油,前期神器
摩托車 快、耗油少、噪音中
皮卡車 大載貨空間=移動倉庫,可衝撞感染者(損耐久)
巴士(進階) 可改裝成移動據點
- 載具需找齊零件修復(引擎、輪胎、電瓶)+ 汽油(可從廢棄車輛虹吸)。
- 噪音會沿路吸引感染者 —— 開車進城 = 高速搜刮但高調。

7.6 潛行系統

  • 蹲行降噪、草叢隱蔽、投擲石頭引開注意、背後無聲處決(近戰刀類)。
  • 「氣味」機制:身上塗抹感染者內臟可短時間混入屍群(致敬 TWD),但流汗/淋雨會失效。

7.7 技能與成長系統(用進廢退 + 技能點雙軌)

  • 熟練度:常跑步→體力上限提升;常做菜→烹飪回復量+10%;常用槍→後座力下降。
  • 技能樹(升級給點)四大分支:
  • 生存:野外知識(辨毒菇)、快速採集、耐寒、輕足(降噪)
  • 戰鬥:近戰專精、槍械專精、弓箭大師、背刺處決
  • 製作:工匠(省材料 20%)、機械師(修車)、化學家(做藥/炸藥)、建築師
  • 社交:交易折扣、領袖魅力(NPC 招募上限+)、談判(避免掠奪者衝突)

7.8 NPC、陣營與交易

  • 倖存者 NPC:可救援並招募回據點,各有專長(醫生/農夫/槍匠/斥候),會自動工作、防守,也要吃喝(增加後勤壓力)。
  • 陣營:
  • 「方舟聚落」:友善商人陣營,用瓶蓋/彈藥當貨幣交易。
  • 「鏽爪幫」:敵對掠奪者,會伏擊玩家、突襲據點,可攻打其營地。
  • 「白衣會」:神祕研究者,劇情陣營,給血清主線任務。
  • 聲望系統:幫助/敵對行為影響各陣營態度與商店價格。

7.9 任務與劇情

  • 主線:追查病毒起源 → 收集三份血清配方 → 潛入研究設施 → 三結局: ① 製成血清廣播配方(犧牲據點被屍潮圍攻)② 獨善其身前往淨區 ③ 留守建立新文明(開放式)。
  • 支線:NPC 個人任務(找失散家人、取回遺物)、陣營任務、地標解謎(保險箱密碼藏在日記)。
  • 動態事件:空投物資(全圖爭奪)、路過的商隊、求救訊號(可能是陷阱)、屍潮遷徙、直升機墜毀。

7.11 難度自適應細節

  • 遊戲天數越多:感染者總量、變異體出現率、屍潮規模緩慢上升 → 前期寬容、後期緊張。
  • 玩家火力越強,掠奪者裝備也升級(防止後期無聊)。
